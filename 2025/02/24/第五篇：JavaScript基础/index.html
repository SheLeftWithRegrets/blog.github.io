性能优化

防抖和节流的区别及应用场景？

防抖(Debounce)：

  • 定义：多次点击同一个按钮，只响应最后一次
  • 应用：search搜索时，用户不断输入值，用防抖来节约请求资源

节流(Throttle)：

  • 定义：高频率触发的事件，指定单位时间内只响应第一次
  • 应用：
    • 鼠标不断点击触发mousedown(单位时间内只触发一次)
    • 监听滚动事件

闭包

什么是闭包？它的作用和问题是什么？

定义：
一个函数内部定义的函数可以访问其外部函数的变量，即使外部函数已执行完毕。

作用：

  • 创建私有变量
  • 延长变量的生命周期
  • 减少全局变量的定义
  • 避免全局变量的污染

内存泄漏问题：

  • 原因：不再使用的内存没有及时释放
  • 解决方法：
    • 及时清理不再需要的引用
    • 避免不必要的全局变量
    • 减少DOM引用
    • 使用弱引用

原型与原型链

请详细说明原型和原型链的概念及关系

构造函数与实例：

  • new一个构造函数会形成它的实例对象
  • 每个构造函数都有prototype原型对象
  • prototype原型对象里的constructor指向构造函数
  • 每个实例对象都有__proto__对象指向原型对象

原型对象：

  • 是一个公共容器
  • 特定类型所有实例对象都可以访问

原型链：

  • 每个对象都可以有一个原型
  • 原型还可以有它自己的原型
  • 属性查找机制：当需要查找一个对象的属性时，如果对象本身没有这个属性，则会沿着原型链向上查找

作用域与变量提升

var、let、const的区别是什么？

声明方式：

  1. var

    • 存在变量提升
    • 可以重复声明
    • 函数作用域

  2. let

    • 不存在变量提升
    • 不可重复声明
    • 块级作用域
    • 暂时性死区

  3. const

    • 声明时必须赋值
    • 不可重复声明
    • 块级作用域
    • 不可修改（对象属性可修改）

浏览器存储

浏览器存储方式有哪些？

  1. Cookie

    • 大小限制4KB
    • 随HTTP请求发送
    • 可设置过期时间

  2. localStorage

    • 永久存储
    • 大小一般为5MB
    • 不随请求发送

  3. sessionStorage

    • 会话期间存储
    • 标签页隔离
    • 大小一般为5MB

  4. IndexedDB

    • 键值对存储
    • 支持索引
    • 异步操作
    • 支持事务
