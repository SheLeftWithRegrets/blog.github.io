CSS基础

选择器的优先级是什么？

选择器优先级从高到低排序:

  1. !important
  2. 内联样式(style)
  3. ID选择器(#id)
  4. 类选择器(.class)
  5. 通配符(*)
  6. 修饰符

CSS3有哪些新特性？

CSS3引入了许多新特性:

  1. 新的选择器
  2. 盒模型的改进
  3. 动画和过渡效果
  4. 渐变效果
  5. 阴影效果

盒模型

Padding和Margin有什么区别？

  • Padding(内边距): 是盒子模型中元素与盒子之间的边距
  • Margin(外边距): 是元素和元素之间的边距，也叫外边距

解释CSS的盒子模型

盒模型包含以下部分:

  1. 内容(content)

    • 由width和height属性定义
    • 由width和height属性定义

  2. 内边距(padding)

    • 由padding属性定义
    • 影响元素尺寸但可选

  3. 边框(border)

    • 由border属性定义
    • 围绕元素的线
    • border宽度影响元素尺寸

  4. 外边距(margin)

    • 由margin属性定义
    • 外边距可选不影响元素本身尺寸
    • 但影响与其他元素的距离

CSS引入方式

CSS有哪些引入方式？各有什么特点？

CSS有三种主要的引入方式:

  1. 内联样式

    • 在元素标签体中使用style属性添加样式

  2. 内部样式

    • 在html文件中使用style标签编写样式

  3. 外部样式

    • 在css文件中编写样式
    • 使用link或@import导入

link和@import的区别

特性 link @import
加载顺序 与页面同时加载 页面加载完才加载
兼容性 所有浏览器支持 CSS2.1提出,低版本可能不支持
功能范围 可定义RSS等其他事务 只能加载CSS
DOM操作 可通过JS操作DOM 不能通过JS操作

CSS选择器

CSS有哪些选择器？

  1. 基础选择器

    • id选择器(#)
    • class选择器(.)
    • 元素选择器(标签名)
    • 后代选择器(空格)
    • 属性选择器([属性名])
    • 子元素选择器(>)
    • 通配符选择器(*)

  2. CSS3新增选择器

    • 结构伪类选择器(:first-child)
    • 否定伪类选择器(:not())
    • 伪元素选择器(::before)

CSS预处理器

使用CSS预处理器有什么优缺点？

优点:

  1. 支持选择器的嵌套
  2. 可定义变量和混合
  3. 支持函数和运算
  4. 支持导入和模块化
  5. 自动处理厂商前缀

缺点:

  1. 需要编译,增加开发复杂性
  2. 调试困难
  3. 可能产生过度设计
  4. 可能产生冗余CSS代码
